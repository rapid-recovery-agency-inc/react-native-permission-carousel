# React Native Permission Carousel

**The goal** of this project is to create a reusable permission carousel component for React Native applications, used across multiple applications within the scope of @rapid-recovery-agency-inc.

## 📦 Getting Started

Clone the following repository:

```bash
git clone https://github.com/rapid-recovery-agency-inc/react-native-permission-carousel.git
```

Add a `.npmrc` file at the root directory with the following content:

```bash
@rapid-recovery-agency-inc:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_TOKEN
```

Make sure to replace `YOUR_TOKEN` with your actual GitHub token with `read:packages` scope.

Install the project's dependencies.

```bash
npm install
```
